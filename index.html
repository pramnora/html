Azure Test Site 01, index.html

First, I had to go to my Windows Azure a/c.; and, then, autorize Azure to point to my GitHub a/c.; in particular, the 'html' folder respository section...

  • http://pmr1.azurewebsites.net/

Second, I open up GitHub for Desktop application; click show the 'html' repository inside of Explorer; then, use Windows Notepad editor to create 2 [.html] files:-

  • html
    • index.html
    • nextpage.html

Third, I open my web browser; and, go to my GitHub online a/c.; surfing to find the 'html' folder repository. Next, click on [Upload] files...select the 2 files to upload from off my local hard drive...can use 'drag and drop' from Exploer window.

Fourth, and, basically everything is all set up. It's now possible to open up a browser window, and, type in the following to display the web site pages.

  • html>
    • http://pmr1.azurewebsites.net/nextpage.html
    • http://pmr1.azurewebsites.net/nextpage.html
