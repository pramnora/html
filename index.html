Azure Test Site 01, index.html

  1. Sign in to Windows Azure a/c. online; and, authorize Azure to point to your GitHub a/c.; in particular, the 'html' folder repository section...

    • http://pmr1.azurewebsites.net

  2. Now, we are going to create 2 .html files inside of GitHub on our local machine.

    Open 'GitHub for Desktop' application on local machine; click 'show the html' repository inside of Explorer; then, use Windows Notepad editor to create 2 [.html] files:-

    • html
      • index.html
      • nextpage.html

  3. Next, we need to reflect the changes made on our GitHub local machine/to our GitHub online a/c.

    Open web browser; go to my GitHub online a/c.; find the 'html' folder repository; next, click on [Upload] files...

    ...select the 2 files to upload from off the local hard drive...using 'drag and drop' from Explorer window directly into GitHub online window.

  4. Now, it's possible to view the 2 web pages you uploaded by appending their file names to the Azure URL ending...

    • html
      • http://pmr1.azurewebsites.net/index.html
      • http://pmr1.azurewebsites.net/nextpage.html

-(NOTE: It's not necessary to go through step 2...create files on the local machine, first;
you could simply choose to create the files directly inside of GitHub online; and, then, proceed to step 4.)-

Alternatively, you can simply surf to the Azure URL address...

  • http://pmr1.azurewebsites.net

...and, from there just manually type in whichever filenames to go at the end of the URL.
