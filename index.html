Today, I learned that it's possible to set up a FREE web site using Microsoft Azure as a server platform.

It is also possible to authorize Azure to link to any folder repository section of my GitHub a/c. in order to serve up web pages directly from there.

For example, say my Azure a/c. has been set to point to the Github 'html' directory containing the following 2 files:

  • html>
    • index.html
    • nextpage.html

Then, opening up the web browser; and, typing in the following URL address...will serve up either of the above files; simply, just list the filename.extension after the URL...

  • html>
    • http://pmr1.azurewebsites.net/nextpage.html
    • http://pmr1.azurewebsites.net/nextpage.html
